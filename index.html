How to create player list.
1) One player per line.
2) Do not include any spaces in the player name.
3) Ranks are formatted like so: Bronze 1 = B1, Gold 1 = G1, etc...
(Do not include LP for higher ranks: Masters = M, GrandMasters = GM, Challenger = C)
4) Main-Role and Off-Role are formatted like so: Top = TOP, Jungle = JNG, Mid = MID, Adc = ADC, Support = SUP.

Example Formatting (Case Sensitive!):
Player1 G3 TOP MID
Player2 S2 JNG ADC
etc...

Known Bugs: If balance meter moves but no teams are displayed simply click import again.
